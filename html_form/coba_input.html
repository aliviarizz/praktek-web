Jika Anda mengklik tombol "kirim", data formulir akan dikirim ke halaman bernama "/action_page.php".

Kesimpulan:

Ketika mengklik tombol reset, data otomatis akan kembali kosong. Tetapi karena inputnya sudah terisi dengan value maka ketika diklik tombol reset, data akan tetap dalam keadaan sama.

Ketika mengklik tombol kirim, data akan terkirim ke halaman bernama "action_page.php".

Selanjutnya kode program ditambahkan sebuah target dan hasil uji cobanya:

  1. Target="_blank": Halaman setelah diklik tombol kirim maka form akan terbuka di tab baru.
  2. Target="_self": Halaman setelah diklik tombol kirim maka form dikirim pada tab yang sama.
  3. Target="_parent": Halaman setelah diklik tombol kirim maka jika dalam iframe, akan membuka di parent frame.
  4. Target="_top": Halaman setelah diklik tombol kirim maka akan membuka halaman penuh.

Menambah sebuah "method", ada dua method yaitu get dan post. Setiap jenisnya ini memiliki perbedaan tersendiri dalam penggunaannya.

Method yang pertama adalah method "get", ketika menambahkan method ini maka data akan ditampilkan di URL.

Method yang kedua adalah method "post", ketika menambahkan method "post" maka data tidak akan tampil atau tersembunyi.
